# Codex Study Project - AI 컨텍스트

## 📋 프로젝트 정보
- **프로젝트명**: codexstudy
- **목적**: Codex CLI 학습 및 실험
- **주요 언어**: Python
- **버전 관리**: Git

## 🎯 Codex에게 바라는 역할

### 1. 코드 작성 시
- Python PEP8 스타일 가이드 준수
- 함수/변수명은 snake_case 사용
- 모든 함수에 docstring 작성
- 타입 힌트(type hints) 적극 활용
- 에러 처리는 try-except로 명확하게

### 2. 코드 리뷰 시
- 보안 취약점 점검 (SQL injection, XSS 등)
- 성능 병목 지점 지적
- 가독성 개선 제안
- 중복 코드 리팩토링 제안

### 3. 문제 해결 시
- 에러 메시지를 분석해 구체적인 원인 설명
- 해결 방법을 단계별로 제시
- 대안 코드 블록 반드시 제공
- 관련 문서/레퍼런스 링크 첨부

### 4. 학습 지원
- 새로운 개념은 예제 코드와 함께 설명
- 복잡한 로직은 주석으로 상세 설명
- 모범 사례(best practice) 우선 제안
